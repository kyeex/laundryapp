## App Name

LaundryApp

## Subtitle / Short Description

Laundry pickup, delivery, tracking, and payment.

## Full Description

LaundryApp helps customers request laundry pickup and delivery from a local
laundromat. Customers can choose laundry services, add special instructions,
track order status, and pay when the final price is ready.

Owners can manage incoming requests, update order status, price orders, assign
pickup and delivery batches, and coordinate drivers.

Drivers can view assigned pickup and delivery batches and update stop status in
the app.
